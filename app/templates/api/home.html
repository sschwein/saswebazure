Congress API (v1 beta)

Overview

This API is currently a collection of bill data from congress.gov. The date range spans from the 103rd congress (1993) to the present, and includes both legislation from the House of Representatives and the Senate.

Endpoint

This API follows a RESTful format. You can access this api through this endpoint: https://sschwein.com/api/congress/v1/
